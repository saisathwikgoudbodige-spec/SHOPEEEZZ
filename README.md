# SHOPEEEZZ

SHOPEEEZZ is a basic e-commerce application project designed for effortless online shopping.

## Overview

The application aims to provide a simple shopping experience where users can browse products, view details, add items to a cart, and place orders securely.

## Features

- Product browsing
- Product details
- Shopping cart
- User login and registration
- Secure checkout
- Order confirmation
- Seller dashboard
- Order management
