# 🚀 SHOPEEEZZ – MERN Stock Trading Simulator

A full-stack MERN application that allows users to explore stocks, simulate trades, manage portfolios, and analyze investment performance.

## 🌐 Live Demo

Frontend: https://shopeeezz.vercel.app

Backend API: https://shopeeezz.onrender.com

---

## ✨ Features

* JWT Authentication & Authorization
* Role-Based Access Control (User/Admin)
* Stock Market Dashboard
* Buy & Sell Trade Simulation
* Portfolio Management
* Transaction History Tracking
* MongoDB Atlas Integration
* Responsive UI Design

---
